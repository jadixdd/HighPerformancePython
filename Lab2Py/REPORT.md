# Отчет по Лабораторной работе 2
## Решение задачи Дирихле для уравнения Лапласа

### 1. Постановка задачи (Вариант 12)

Требуется решить уравнение $\Delta u = 0$ на сетке.

**Граничные условия:**
* $u(0, y) = 5y - y^2$
* $u(1, y) = 4 - y^2 + 5y$
* $u(x, 0) = x^2 + 3x$
* $u(x, 1) = x^2 + 3x + 4$
